# SeldonFrame Blocks

This directory contains example BLOCK.md specs that can be implemented using AI coding agents.

## Quick Start

1. Copy a BLOCK.md file from `examples/`
2. Open your SeldonFrame project in Windsurf, Cursor, or Claude Code
3. Prompt: "Read `blocks/examples/courses.block.md` and implement this block following existing codebase patterns"
4. Review generated files and run migrations/build

## Create Your Own

Read the full specification in [`BLOCK_MD_SPEC.md`](../BLOCK_MD_SPEC.md).

## Marketplace

Pro Agency users can submit BLOCK.md specs to the native SeldonFrame marketplace.
