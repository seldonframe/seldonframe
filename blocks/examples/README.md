# BLOCK.md Examples

These examples show the universal BLOCK.md format used for marketplace and custom block generation.

## Included

- `courses.block.md` — full, multi-entity reference example
- `invoicing.block.md` — smaller example suitable for quick testing

## Usage

1. Pick an example BLOCK.md
2. Ask your AI coding agent to implement it in your SeldonFrame repo
3. Run migration/build and validate behavior

All examples are framework-agnostic by design.
