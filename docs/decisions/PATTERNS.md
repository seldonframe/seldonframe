# PATTERNS.md — Max's known tendencies (loaded by every /reflect)

Seeded 2026-07-15 from documented history, not speculation. The loop NAMES a
pattern in the answer card when it fires. Corrections welcome — a wrong pattern
is worse than none.

- **Estimates run long.** Documented: the queue-migration class ("one weekend"
  → three). Rule: 2x any time estimate until the log proves otherwise.
- **Building displaces selling.** The only motion that ever produced revenue is
  live-selling, yet build-work always wins the calendar (10-close sprint open
  since 07-08 while ~15 features shipped). When a decision trades build vs
  sell, weight sell.
- **Features ship dark and stay dark.** 15+ flags on main, several 2+ weeks
  unflipped (07-15 census). A "we'll flip it after X" plan needs an owner and a
  date or it won't happen.
- **Aesthetic decisions get re-litigated.** teal → emerald → forest in ~24h
  (07-13). One extra day of sitting with a design call saves two rework cycles.
- **Side-quests bloom mid-sprint.** New skills/repos/content engines appear at
  a high rate and are individually excellent — but §5 of the wedge doc froze
  "new surfaces" and the freeze has no enforcement. Name it when it fires.
- **Strengths to lean on (also patterns):** verify-before-done discipline is
  real (smokes, receipts, never-lies catches); comfort with killing his own
  ideas when shown evidence; asks for honesty and rewards it — do not soften
  calls to be nice.
