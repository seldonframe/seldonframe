# BLOCK: OG Image Generator

**Description**  
Generate branded Open Graph images for campaign, landing, and shareable links across the workspace.

**Trigger Phrases**  
- "Create OG images for my landing pages"
- "Generate branded social preview images"

**Behavior**  
Create reusable Open Graph image variants for key pages, offers, and campaigns so every shared link has a polished branded preview. Apply workspace branding tokens for layout, background, typography, and CTA emphasis while keeping templates easy to reuse for rapid launches. Favor actionable patterns such as which page types need dedicated preview images, where visual consistency improves campaign readiness, and how faster asset generation supports launch velocity.
