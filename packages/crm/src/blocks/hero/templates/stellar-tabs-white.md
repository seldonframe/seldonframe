# stellar-tabs-white

**Vibe:** white background, AI-tool confidence. Two-line headline where
the second line renders in dark-to-gray gradient text. Below the CTA:
a 4-tab switcher (Intake → Schedule → Convert → Deliver) that
auto-cycles every 4s, each tab showing a different product mockup card
on a soft indigo gradient backdrop. The 4 tabs preview the four phases
of every SeldonFrame workspace, so the hero doubles as a product
demo.

**When to use:** AI workspace platforms, multi-feature SaaS,
productivity suites. Anywhere the brand wants to communicate "we do
four things well" upfront.

**Copy pattern:** Two-clause headlines that split naturally at the
midpoint. The `shinyWord` (if provided) becomes the split point — the
shiny word and everything after it gets the gradient treatment. Use
proofTile to render a "4.9 rating from 18.3k users" pill above the
headline (most natural badge for this template).

**Headline examples (good):**
- "Work Smarter. Move Faster. AI Powers You Up." + shinyWord "AI"
- "Stop Chasing Leads. Start Closing." + shinyWord "Start"
- "Build the System Once. Run It Forever." + shinyWord "Run"

**Consumes:** kicker OR proofTile (for the badge — proofTile wins if
both set), headline, shinyWord, subheadline, ctaText, ctaLink,
secondaryCta. Tab content is hardcoded — no copy needed for it.
